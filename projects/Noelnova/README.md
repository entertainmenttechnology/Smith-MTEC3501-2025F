# The Contagion Zone

A choice-driven, narrative-focused Twine project exploring survival, morality, and societal collapse through interactive storytelling.

## üìò Project Overview
**The Contagion Zone** is an emotionally intense Twine game set in a quarantined region on the brink of collapse. Players navigate a world infected not only by disease, but by fear, secrecy, and the breakdown of social order. Every choice shapes the story, relationships, and ultimately the fate of the world.

This repository contains the Twine source files, documentation, and assets for the project.

---

## üéÆ Key Features
- **Branching Narrative:** Every decision impacts the storyline and leads to unique paths and endings.
- **Speculative Themes:** Explores trust, systemic collapse, and the blurred line between humanity and monstrosity.
- **Emergent Storytelling:** Player choices create individualized journeys.
- **Atmospheric Design:** Includes original artwork, sound integration, and thematic writing.

## üöÄ Getting Started
### 1. Clone the repository
```
git clone https://github.com/USERNAME/ContagionZone.git
```

### 2. Open the Twine project
Open Twine 2 and import the story file from `/twine-story-files`.

### 3. Run the game
Use **Play** in Twine to test interactive passages.

### 4. Export builds
Go to **Publish to File** to create the HTML version for deployment.

## üß≠ Narrative Summary
At its heart, **The Contagion Zone** challenges players with difficult moral decisions during a societal breakdown. The story examines:
- Collapse of trust
- Fragility of order
- Moral ambiguity
- Uncovering hidden truths

The climax forces players to choose between self-sacrifice and unleashing the plague to save loved ones, producing multiple endings that reveal different facets of truth and humanity.
